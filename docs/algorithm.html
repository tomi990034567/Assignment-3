The Algorithm

Curating Data and Designing Algorithms
Visit GitHub Page

In the digital age, university websites play a crucial role in providing students with easy access to academic resources. One of the fundamental challenges faced by students is sorting through a vast array of literature. To address this issue, we have developed the Algorithm. The algorithm allows students to search through a database of academic literature relevant to various courses and subjects at the university. When students access the literature search page, they are presented with a number of filter options to refine their results. Only the records that meet all the specified filter conditions are returned as results. This allows students to efficiently narrow down the literature database to find the most relevant resources for their work.

The Features

The algorithm aims to make this course literature list more accessible, by being able to easily search:

  • Author
  • Keyword
  • Book
  • Publishing year
