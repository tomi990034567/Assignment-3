The Dataset

Curating Data and Designing Algorithms
done by Group 4 Visit GitHub Page.

For this group project assignment, we curated an original dataset, then designed an algorithm to analyze that dataset. The goal was to gain experience finding and preparing real-world data, as well as conceptualizing how that data could be processed algorithmically to address a specific problem or need. We then published our dataset, algorithm description, and this short explanatory website on GitHub for review. Through this collaborative experience, we improved our data science skills.

The Assignment

Things we considered in our decision-making process include:

  • Which dataset?
  • Which problem will we solve?
  • The actual use case?
  • The depth of the dataset?
  • Features of the algorithm?
  • Who will use the algorithm?
